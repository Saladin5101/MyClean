MyClean:一款适用于macOS15及以上版本系统的缓存清理程序

该软件是一款为了保持Mac设备更快、更稳定的运行(虽然这个程序一点也不稳定)而编写的一个应用程序与缓存清理程序,并欢迎各界人士提出相关意见

重要说明:关于致命缺陷
该软件由于一些不可明说的原因(暗说也不行!)暂时无法正常运行,目前正在努力修复该问题
